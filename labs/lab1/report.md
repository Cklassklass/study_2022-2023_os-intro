# 3. Теоретическое введение

Виртуальная машина (ВМ) — это программная эмуляция физического компьютера, работающая как изолированная система с собственной ОС внутри «хоста» (вашего ПК).

Linux — это семейство свободных, Unix-подобных операционных систем с открытым исходным кодом, основанных на ядре Linux, разработанном Линусом Торвальдсом в 1991 году. В отличие от Windows, Linux поставляется в виде различных дистрибутивов (Ubuntu, Debian, Fedora)

Fedora — это популярный, современный дистрибутив Linux, спонсируемый Red Hat
 Она ориентирована на использование свободного ПО, инновации, быстрые 
обновления программ и внедрение новых технологий. Fedora часто служит 
полигоном для тестирования решений, которые позже попадают в 
коммерческие системы.
