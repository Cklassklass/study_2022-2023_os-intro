# Теоретическое введение

**vi** — это классический текстовый редактор, который есть практически в любой UNIX-подобной системе (Linux, macOS, BSD). Его главная особенность — **модальность**: клавиатура работает по-разному в зависимости от того, в каком режиме вы находитесь.

Основные режимы работы

1. **Командный режим (Command Mode)**: активен по умолчанию. Здесь нажатия клавиш — это команды (удаление строки, перемещение курсора, поиск).
2. **Режим вставки (Insert Mode)**: для ввода текста. Чтобы войти в него, нажмите i. Чтобы вернуться в командный режим, нажмите ESC.
3. **Режим последней строки (Last-line Mode)**: для системных команд (сохранение, выход). Активируется нажатием : в командном режиме
